cURL es un software orientado a la transferencia de archivos sobre protocolos de red HTTP y FTP entre otros. cURL suele venir de forma nativa en las distribuciones basadas en GNU/Linux y es Open Source. En PS cURL es un alias de Invoke-WebRequest, podemos comprobarlo con el comando Get-Alias.

PS> Get-Alias curl

CommandType     Name                                               ModuleName
-----------     ----                                               ----------
Alias           curl -> Invoke-WebRequest

PS> curl http://services.groupkt.com/country/get/all | ConvertFrom-Json | Select name
